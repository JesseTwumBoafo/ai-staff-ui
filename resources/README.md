# App icon sources

`icon.png` is the 1024 px source electron-builder converts to the macOS `.icns`
at package time (`build.mac.icon` in `package.json`).

Note: the current `icon.png` is a placeholder. It was upscaled from the existing
256 px `electron/icon.ico` frame, so it is soft at full size. Replace it with a
true 1024 px master (from Bubble or Dani) before any public macOS release.
